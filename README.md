# Docker installation for Ubuntu

## Install latest Docker version
```
curl -fsSL https://github.com/thuanpn19/Docker-Installer/blob/main/install-docker.sh | bash
```
## Install specific version
```
version="20.10.4"
curl -fsSL https://github.com/thuanpn19/Docker-Installer/blob/main/install-docker.sh | bash
```

## Uninstall Docker
```
curl -fsSL https://github.com/thuanpn19/Docker-Installer/blob/main/uninstall-docker.sh | bash
```
